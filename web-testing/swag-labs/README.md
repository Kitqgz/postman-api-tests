# Swag Labs Testing Project

## Описание проекта

Swag Labs — учебное веб-приложение для тестирования интернет-магазина.

## Цель проекта

Проверить корректность работы функционала авторизации пользователей.

## Документация

- Test Plan
- Checklist
- Test Cases
- Bug Reports
- Test Summary Report

## Используемые инструменты

- Google Chrome
- Chrome DevTools
- GitHub
